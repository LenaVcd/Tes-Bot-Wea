# Tes-Bot-Wea
Simple Bot Wea

### FOR TERMUX USER
```bash
> pkg update && pkg upgrade
> pkg install git
> pkg install nodejs
> pkg install ffmpeg
> pkg install imagemagick
> git clone https://github.com/LenaVcd/Tes-Bot-Wea
> cd Tes-Bot-Wea
> npm install
```
###### Run
```bash
> node . [<session name>] (session name is optional)
```
